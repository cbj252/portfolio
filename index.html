About

This is the final project from The Odin Project, where I learnt web programming. It is a recreation of Facebook, allowing for messaging between users.

Like Facebook, users can add other users as friends. They can post messages, see their friends' messages and upload profile pictures. The messages are stored in a MongoDB database, so messages will be kept through server refreshes. The site is also responsive and will change it's layout to fit smaller screens like mobile phones or tablets.

About

This is a forum template. Users can register and create threads and posts, and edit their own posts. Only administrators and owners can delete posts. Using an admin portal only visible for admins and owners, admins can be added or removed if you have the permissions for it.

This app was made using React for the frontend making calls to an Express API, which quickly taught me the limitations of the fetch api and asynchronus code. It also gave me an opportunity to learn about transferring data via json between my frontend and backend. The login page in the frontend is made using web tokens and cookies, so users can stay logged in while keeping the backend stateless.
